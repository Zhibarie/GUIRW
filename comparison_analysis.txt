ANALISIS PERBANDINGAN: rwGUI vs Rusted Warfare Modding Reference
================================================================================

SEKSI YANG ADA DI REFERENSI TAPI TIDAK ADA/TIDAK LENGKAP DI rwGUI:
================================================================================

1. [global_resource_NAME] - TIDAK ADA DI rwGUI
   - Referensi memiliki section khusus untuk global_resource_NAME (baris 3164)
   - rwGUI hanya memiliki "resource" untuk local resource
   - Field yang hilang: priority, displayColor, displayRoundedDown, 
     displayTextPrefix, displayTextPostfix, displayTextAppendResource,
     displayWhenZero, displayPos, iconImage, iconImageUseInText,
     displayNameHideWhenIconShownInText, displayNameHideWhenIconShownInHUD,
     displayColorUseInText

2. [arm_#] - TIDAK ADA SEBAGAI SEKSI TERPISAH
   - Referensi menyebutkan [leg_#] / [arm_#] sebagai section terpisah (baris 1530)
   - rwGUI hanya memiliki "leg" tanpa dukungan eksplisit untuk "arm"
   - Meskipun strukturnya mirip, seharusnya ada opsi untuk arm sebagai alternatif

3. [hiddenAction_NAME] - TIDAK ADA
   - Referensi menyebutkan [action_NAME] / [hiddenAction_NAME] (baris 1748)
   - rwGUI hanya memiliki "action" tanpa opsi "hiddenAction"
   - hiddenAction adalah action yang tidak ditampilkan di UI

4. [comment_NAME] - TIDAK ADA
   - Referensi memiliki section comment_NAME (baris 3433)
   - Ini digunakan untuk menambahkan komentar dalam file INI
   - Tidak ada di rwGUI

5. [template_NAME] - TIDAK ADA
   - Referensi memiliki section template_NAME (baris 3441)
   - Template digunakan dengan @copyFromSection, @define, @global, @memory
   - File .template tidak dapat dibuat/diedit di rwGUI

6. [action_spawnedInitialize] - TIDAK ADA
   - Referensi memiliki section khusus action_spawnedInitialize (baris 3472)
   - Ini adalah action khusus yang dijalankan saat unit di-spawn
   - Tidak ada di rwGUI

FILE KHUSUS YANG TIDAK DIDUKUNG:
================================================================================

1. mod-info.txt - Hanya parsing basic fields (title, description, tags, 
   minVersion, id, requiredMods, requiredModsMessage)
   - Missing: thumbnail, sourceFolder, dan lainnya

2. all-units.template - Tidak ada dukungan
3. *.template files - Tidak ada dukungan
4. .copyfrom files - Hanya diparse tapi tidak ada editor khusus

KESIMPULAN:
================================================================================

rwGUI adalah tool yang sangat BASIC dibandingkan dengan referensi lengkap.
Tool ini hanya mendukung subset kecil dari semua fitur modding Rusted Warfare.

Section yang SEPENUHNYA HILANG:
1. global_resource_NAME
2. arm_# (sebagai section terpisah)
3. hiddenAction_NAME
4. comment_NAME
5. template_NAME
6. action_spawnedInitialize

Field yang hilang: RATUSAN field dari referensi tidak ada di rwGUI

File khusus yang tidak didukung:
- all-units.template
- *.template files
- Banyak field di mod-info.txt

rwGUI cocok hanya untuk modding UNIT DASAR, bukan untuk modding advanced
yang membutuhkan fitur-fitur seperti shields, energy systems, templates,
global resources, dll.
